# 🌐 Portfolio Website (Full Stack)

A full-stack portfolio website built to showcase projects, skills, and experience with a functional contact system.

---

## 🚀 Features

- Responsive frontend UI (HTML, CSS, JavaScript)
- Contact form with backend integration
- Data stored in MySQL database
- Email notification using Spring Boot
- Clean and organized code structure

---

## 🛠 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Spring Boot
- REST API

### Database
- MySQL

### Tools
- Git & GitHub

---

## 📩 Contact Flow

User Form → Spring Boot API → MySQL Database + Email Notification

---

## ⚙️ Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/Raza084/portfolio-website.git
cd portfolio-website

### 2. Set environment variables
export DB_PASSWORD=your_database_password
export EMAIL_PASSWORD=your_app_password

### 3. Run backend
cd backend/portfolio
./mvnw spring-boot:run

### 3. Open frontend
Open:
frontend/index.html
